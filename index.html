HTML Dersi

Web'in İskeletini Öğrenin

HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan dildir.

  • Tag'ler: <açılış> içerik </kapanış>
  • Attribute'lar: Tag'lere ek özellik: <tag özellik="değer">
  • Semantik HTML: Anlamlı tag'ler kullan (header, nav, section)
HTML 
<!DOCTYPE html>
<html lang="tr">
<head>
    <meta charset="UTF-8">
    <title>Benim Sayfam</title>
</head>
<body>
    <header>
        <h1>Merhaba Dünya!</h1>
        <nav>
            <a href="#anasayfa">Ana Sayfa</a>
            <a href="#hakkimda">Hakkımda</a>
        </nav>
    </header>
    
    <main>
        <section id="anasayfa">
            <h2>Hoş Geldiniz</h2>
            <p>Bu benim ilk web sayfam.</p>
            <img src="resim.jpg" alt="Açıklama">
        </section>
    </main>
    
    <footer>
        <p>© 2024 Benim Şirketim</p>
    </footer>
</body>
</html>

CSS Dersi

Web'e Stil Verin

CSS Nedir?

CSS (Cascading Style Sheets), HTML'e stil veren dildir.

  • Selector'lar: Hangi elementi seçeceğiz?
  • Property-Value: Hangi özelliği ne değere çevireceğiz?
  • Box Model: Content, Padding, Border, Margin
  • Flexbox & Grid: Modern layout sistemleri
CSS 
/* CSS Reset - Tüm elementleri sıfırla */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* Body stilleri */
body {
    font-family: 'Poppins', sans-serif;
    background-color: #f5f7fa;
    color: #333;
    line-height: 1.6;
}

/* Container - İçeriği ortala ve sınırla */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
}

/* Buton stilleri */
.btn {
    display: inline-block;
    padding: 12px 24px;
    border: none;
    border-radius: 5px;
    font-size: 16px;
    font-weight: 600;
    cursor: pointer;
    transition: all 0.3s ease;
}

.btn-primary {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
}

.btn-primary:hover {
    transform: translateY(-2px);
    box-shadow: 0 10px 20px rgba(102, 126, 234, 0.3);
}

/* Flexbox örneği */
.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
    gap: 20px;
}

/* Media Query - Mobil uyumluluk */
@media (max-width: 768px) {
    .container {
        padding: 0 15px;
    }
    
    .flex-container {
        flex-direction: column;
    }
}

JavaScript Dersi

Web'i Interaktif Yapın

JavaScript Nedir?

JavaScript, web sayfalarını interaktif yapan programlama dilidir.

  • Değişkenler: let (değişebilir), const (sabit)
  • Fonksiyonlar: Tekrar kullanılabilir kod blokları
  • DOM Manipülasyonu: HTML elementlerini değiştirme
  • Event'ler: click, hover, input gibi kullanıcı hareketleri
JavaScript 
// 1. DEĞİŞKENLER - Veri saklamak için
let kullaniciAdi = "Ahmet";  // Değiştirilebilir
const PI = 3.14;            // Sabit (değiştirilemez)
var eskiYontem = "Kullanma"; // ESKI - artık kullanma

// 2. VERİ TİPLERİ
let metin = "Merhaba";      // String
let sayi = 42;              // Number
let mantik = true;          // Boolean
let liste = ["Elma", "Armut", "Muz"]; // Array
let kisi = {                // Object
    ad: "Ali",
    yas: 30,
    sehir: "İstanbul"
};

// 3. FONKSİYONLAR
function selamVer(isim) {
    return "Merhaba " + isim + "!";
}

// 4. ARROW FUNCTION (Modern)
const topla = (a, b) => {
    return a + b;
};

// 5. DOM MANİPÜLASYONU
// Element seçme
const baslik = document.getElementById("baslik");
const buton = document.querySelector(".btn-primary");

// Element değiştirme
baslik.textContent = "Yeni Başlık";
baslik.style.color = "red";

// Event listener ekleme
buton.addEventListener("click", function() {
    alert("Butona tıklandı!");
    baslik.style.transform = "scale(1.1)";
});

// 6. KONSOL KULLANIMI (Debug)
console.log("Merhaba konsol!");
console.log("Kullanıcı adı:", kullaniciAdi);
console.table(liste);
